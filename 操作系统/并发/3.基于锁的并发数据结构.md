# 基于锁的并发数据结构

## 1.并发计数器

* 一个简单的计数器: 可扩展性差
* 可扩展的计数器: 懒惰计数器（一个全局计数器和多个局部计数器（每个CPU核一个））
  * 一个全局锁，每个局部计数器一个锁
  * 局部计数器的值定期转移到全局计数器，取决于阈值
